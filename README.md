# 📊 Data Analysis Project

## Overview

This project demonstrates a complete end-to-end data analysis workflow, starting from raw data and ending with business insights presented through an interactive dashboard and presentation. The project focuses on data preparation, exploratory analysis, SQL querying, visualization, and reporting to support data-driven decision-making.

## Dataset

The dataset contains structured business data used for analysis. It was imported into Python for preprocessing and exploratory data analysis before being loaded into PostgreSQL for advanced querying.

## Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Examined data structure and column information.

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distribution and summary statistics.
- Identified trends, patterns, and potential anomalies.
- Created visualizations to understand relationships between variables.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data formats.
- Prepared the dataset for analysis.

### 4. SQL Analysis
- Imported the cleaned dataset into PostgreSQL.
- Wrote SQL queries to answer business questions.
- Used filtering, grouping, joins, aggregate functions, and sorting for analysis.

### 5. Dashboard Development
- Connected Power BI to the processed data.
- Built an interactive dashboard with KPIs and visualizations.
- Added filters and slicers for better user interaction.

### 6. Reporting
- Summarized key findings in a structured report.
- Created a professional presentation using Gamma to communicate insights effectively.

## Dashboard

The Power BI dashboard includes:

- Trend analysis
- Category-wise performance

- Charts and visual comparisons
- Business insights for decision-making
